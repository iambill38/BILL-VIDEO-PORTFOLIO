# BILL-VIDEO-PORTFOLIO


A premium cinematic videography and creative direction portfolio website. Dark, modern aesthetic with smooth animations and glassmorphism effects.

## Features

- **Secret Code Access** - Password protected entry (code: "unlocked")
- **Responsive Design** - Mobile, tablet, and desktop optimized
- **Animated Background** - Floating particle effects and gradients
- **Multiple Pages** - Home, About, Photography, Welcome
- **Contact Integration** - Direct email links to christiankongolo35@gmail.com
- **Filter Gallery** - Photography archive with category filters
- **Lightbox Viewer** - Click to expand gallery images
- **Lucide Icons** - SVG-based professional iconography

## Pages

- **welcome.html** - Secret code entry page
- **index.html** - Main portfolio with showreel, reels, music videos
- **about.html** - Bio, stats, skills, values, and core competencies
- **photography.html** - Filterable photography gallery with lightbox

## Tech Stack

- HTML5
- CSS3 (Glassmorphism, animations, responsive grid)
- Vanilla JavaScript
- Lucide Icons
- Google Fonts (Space Grotesk, Inter)

## Color Scheme

- Background: #050505 (pure black)
- Cards: #101010 (dark gray)
- Text: #ffffff (white)
- Accent: #c0c0c0 (chrome/silver)
- Secondary: #a1a1aa (light gray)

## Quick Start

1. Open `welcome.html` in browser
2. Enter secret code: **unlocked**
3. Explore the portfolio

## Customization

**Update Bio** - Edit text in `about.html`

**Change Email** - Find `christiankongolo35@gmail.com` and replace with your email

**Add Videos** - Replace YouTube video IDs in index.html

**Modify Colors** - Edit CSS variables in `:root` section

**Update Social Links** - Modify nav links in navigation sections
